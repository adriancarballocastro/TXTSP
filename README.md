# TXTSP – Text Secure Protocol

TXTSP (Text Secure Protocol) is the ultimate protocol for securely and privately sharing .txt files. Keep your documents protected while sending them anywhere.

## Features

Secure encryption: Protect your .txt files from unauthorized access.

Lightweight and fast: Optimized for text files, no complications.

Full compatibility: Works with any standard .txt file.

Easy integration: Perfect for projects or scripts that need to share text securely.

## Installation
### Clone the repository
git clone https://github.com/tuusuario/TXTSP.git

### Enter the directory
cd TXTSP

### Run the server
java -cp out/production/TXTSP Server  

### Run the client
java -cp out/production/TXTSP Client
