# TXTSP – Text Secure Protocol

TXTSP (Text Secure Protocol) es el protocolo definitivo para compartir archivos .txt de forma segura y privada. Mantén tus documentos protegidos mientras los envías a cualquier parte.

## Características

Cifrado seguro: Protege tus archivos .txt de accesos no autorizados.

Ligero y rápido: Optimizado para archivos de texto, sin complicaciones.

Compatibilidad total: Funciona con cualquier archivo .txt estándar.

Fácil integración: Perfecto para proyectos o scripts que necesiten compartir texto de manera segura.

## Instalación
### Clona el repositorio
git clone https://github.com/tuusuario/TXTSP.git

### Entra en el directorio
cd TXTSP

### Ejecutar servidor
java -cp out/production/TXTSP Server  

### Ejecutar cliente
java -cp out/production/TXTSP Client
